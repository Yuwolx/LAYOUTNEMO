# 🗺 Roadmap

LAYOUTNEMO의 현재 상태와 앞으로의 방향을 기록합니다.
이 서비스는 **완결된 결과물이 아니라 실험과 개선의 출발점**이므로, 이 로드맵도 상황에 따라 유연하게 조정됩니다.

---

## ✅ 완료

- [x] 데모 버전 구현 (2025-12-21)
- [x] 도메인 구매 + Vercel 배포 (`layoutnemo.com`)
- [x] 구글 소셜 로그인 (Supabase Auth)
- [x] 블로그 구조 재설계 + 다크 모드 적용 (2026-04-23)

## 🚧 진행 중 / 내부 개선

- [ ] 시급도 단계/라벨 **통일** (현재 코드 내 라벨 불일치)
- [ ] 데이터 구조 설계 vs 구현 갭 **정리** (공식 폐기 또는 재설계)

## 📝 예정 (우선순위 조정 중)

- [ ] **협업 기능** — 공유 링크, 실시간 동기화
- [ ] **모바일 앱**
- [ ] **블럭 템플릿** — 반복되는 블럭 패턴을 저장/불러오기
- [ ] **반복 업무 자동 생성** — 주기적 블럭
- [ ] **통계 및 인사이트** — 시급도 분포, 결별 비율 등

## 🧭 방향성

- **"사고 공간"이라는 개념 자체를 계속 검증**
- 사용 경험을 바탕으로 **기능 확장과 구조 개선을 동시에** 탐색
- **로컬 우선, 사용자 통제감**을 해치지 않는 선에서만 AI 확장
