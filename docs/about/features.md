# 주요 기능

## 🧱 블럭 관리

- **자유로운 배치** — 드래그 앤 드롭으로 캔버스 어디든 배치
- **자동 연결** — 블럭을 가까이 놓으면 자동으로 관계선 생성
- **선 클릭 삭제** — 관계선을 클릭하면 연결 해제
- **블럭 복사** — Alt/Option 키를 누르고 클릭하여 빠른 복사
- **블럭 완료** — 완료 영역으로 드래그하면 완료 처리, 언제든 복원 가능

## 🤖 AI 기능 (선택적)

- **스마트 블럭 생성** — 자연어로 입력하면 제목, 기한, 시급도를 자동 추출
- **정리하기** — 여러 블럭을 살펴보고 AI가 배치/연결/시급도에 대한 제안을 카드 형태로 하나씩 보여줍니다. 적용/건너뛰기를 직접 선택하세요.
- **언제든 토글** — 헤더의 AI 버튼으로 ON/OFF. AI를 꺼도 모든 기능은 수동으로 사용할 수 있습니다.

## 🗂 다중 캔버스

- 프로젝트별로 **여러 작업 공간**을 만들어 분리
- `Cmd/Ctrl + K` 로 빠르게 전환
- 각 캔버스마다 독립적인 블럭과 결 관리

## ⌨️ 키보드 단축키

| 단축키 | 기능 |
|--------|------|
| `Cmd/Ctrl + N` | 새 블럭 만들기 |
| `Cmd/Ctrl + Z` | 되돌리기 (최대 50단계) |
| `Cmd/Ctrl + K` | 캔버스 선택 |
| `Cmd/Ctrl + F` | 블럭 검색 |
| `Cmd/Ctrl + E` | 데이터 내보내기 |
| `Alt/Option + 클릭` | 블럭 복사 |
| `Esc` | 다이얼로그 닫기 |

## 🔍 검색 및 필터

- 제목/내용으로 **블럭 검색** — 결과 클릭 시 해당 블럭 강조
- **시급도별 필터링**
- **기한 초과 블럭만 보기**
- 활성 필터가 있을 때는 시각적 표시

## 💾 데이터 관리

- **자동 저장** — 30초마다 자동으로 저장
- **되돌리기** — 최대 50단계까지
- **휴지통** — 삭제한 블럭 10개까지 보관, 복원 가능
- **내보내기** — JSON 또는 Markdown 형식 지원

## 🏷 결(Facet) 관리

- **커스텀 결 생성** 및 색상 지정
- 결 추가/수정/삭제
- 결을 클릭하면 해당 결에 속한 블럭들이 캔버스 위에서 강조됩니다.

## 🌙 다크모드

- **자동 다크모드** 지원 (시스템 설정 따라감)
- 부드러운 전환 애니메이션

## 🔐 로컬 우선, 클라우드 선택

- 기본적으로 **모든 데이터는 로컬 브라우저에 저장**됩니다. 로그인 없이 바로 사용할 수 있어요.
- 원하는 경우에만 **구글 계정으로 로그인**하여 여러 기기에서 동기화할 수 있습니다.
