# About LAYOUTNEMO

LAYOUTNEMO는 **네모난 블럭을 캔버스에 펼쳐놓는** 사고 공간입니다. 이름부터 그렇습니다 — **LAYOUT (펼쳐놓다) + NEMO (네모)**. 이름 안에 이미 이 서비스의 본질이 담겨 있어요.

이 섹션에서는 LAYOUTNEMO가 왜 만들어졌는지, 무엇으로 구성되어 있는지, 어떤 기능을 제공하는지, 그리고 어떤 기술로 만들었는지를 다룹니다.
